# Homepage

## Installation

```shell
npm i
```

## Development

-   `npm start` - Development server
-   `npm run build` - Build production to `dist/`
-   `npm run eslint` - Run eslint pass
-   `npm run stylelint` - Run stylelint pass

## Deployment

-   `npm run deploy` - Build prod release and deploys to server
